# oa

若干SSM的办公OA管理系统
====

#### 技术： SSM  +   layUI  +   JQuery  +  mysql 


# 项目介绍

本项目是一个基于SSM框架的办公OA管理系统 ，系统所拥有的模块如下： 
1.登录模块
2.我的工具
3.部门管理
4.员工管理
5.专业管理
6.班级管理
7.学生管理
